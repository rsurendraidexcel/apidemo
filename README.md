#### This Appliction usee

- JavaScript Runtime Nodejs
- Typescript
- Express
- sequelize ORM
- Mysql Database


#### To Run the Program

### This command will Build project
````
npm run build

````
#### This command will Run in Dev Mode

````
npm run dev

````
#### This command will Run in production Mode

````
npm run start

````

#### This command will run the local db migration

````
npm run migrate

````
